What is a dataset? A dataset is any structured or unstructured collection of information that is processed, stored, or transmitted by the organisation — including databases, spreadsheets, file shares, application data, backups, and logs. Under GDPR Art. 4(1), any dataset containing information relating to an identified or identifiable natural person constitutes personal data processing.
Does this dataset contain any PII?
Personally Identifiable Information (PII) is any data that could directly or indirectly identify a natural person — names, email addresses, IP addresses, financial records, health data, etc.
